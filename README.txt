TOMBS

Packing Project:
	1) Open the package "app"
	2) In the path line write cmd
	If package "app" contains package build, then write in the console: gradle clean
	3) Write in the console: gradle customJar

Run Project:
	1) After, go to the package "app" -> "build" -> "libs"
	2) In the path line write cmd
	3) In the console, write java -jar TombsApp.jar
